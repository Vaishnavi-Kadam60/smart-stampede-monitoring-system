**Smart Stampede Monitoring & Prevention System**
Overview

This project is an AI-based crowd monitoring system designed to detect overcrowding and help prevent stampedes using real-time video analysis.

The system uses:

YOLOv8
OpenCV
Raspberry Pi 4
Python

**When crowd density exceeds a threshold:

Email alerts are sent
Raspberry Pi is triggered
LED/Buzzer alerts can activate**


Features
       ** Real-time people detection
        Crowd counting
        Threshold-based alert system
        Email notifications
        Raspberry Pi integration
        Low-cost smart monitoring solution**
